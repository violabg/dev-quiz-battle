# Theme Configuration

## Color System in OKLCH

All theme colors use OKLCH (Oklch Lightness Chroma Hue) format.

### Primary Colors

- `--color-primary: oklch(0.65 0.15 260)` - Blue
- `--color-primary-light: oklch(0.75 0.10 260)` - Light blue
- `--color-primary-dark: oklch(0.50 0.20 260)` - Dark blue

### Semantic Colors

- `--color-success: oklch(0.65 0.15 150)` - Green
- `--color-warning: oklch(0.75 0.18 70)` - Yellow/Orange
- `--color-error: oklch(0.65 0.22 30)` - Red
- `--color-info: oklch(0.70 0.12 260)` - Blue

### Neutral Colors

- `--color-text-primary: oklch(0.25 0.02 0)` - Dark gray/black (light mode)
- `--color-text-secondary: oklch(0.60 0.02 0)` - Medium gray
- `--color-bg: oklch(0.99 0.00 0)` - White (light mode)
- `--color-bg-secondary: oklch(0.95 0.01 0)` - Light gray

## Light Mode Palette

- Background: White (`#FFFFFF`)
- Text: Dark gray (`#1F2937`)
- Borders: Light gray (`#E5E7EB`)
- Cards: White with shadow

## Dark Mode Palette

- Background: Dark gray (`#111827`)
- Text: Light gray (`#F3F4F6`)
- Borders: Dark gray (`#374151`)
- Cards: Darker gray with subtle border

## Typography

- **Headings**: font-bold, text-gray-900/white
- **Body**: font-normal, text-gray-700/gray-300
- **Labels**: font-semibold, text-gray-600/gray-400
- **Code**: font-mono, text-gray-800/gray-200
